# Panduan Penggunaan Jaffran Prewed Theme

## Langkah-Langkah Aktivasi

1. **Aktivasi Tema**
   - Login ke WordPress Admin Dashboard
   - Pergi ke `Appearance > Themes`
   - Cari "Jaffran Prewed Theme"
   - Klik "Activate"

2. **Mengatur Konten**
   - Pergi ke `Appearance > Customize`
   - Anda akan melihat beberapa section yang dapat dikustomisasi

## Customizer Settings

### 1. Hero Carousel
**Lokasi**: Appearance > Customize > Hero Carousel

- **Slide 1, 2, 3**: Setiap slide memiliki:
  - Gambar (Upload gambar 1920x1080px)
  - Judul (Teks besar di tengah)
  - Subjudul (Teks kecil di bawah judul)

- **Tombol CTA**:
  - Teks Tombol (default: "Hubungi Kami")
  - Link Tombol (URL tujuan)

**Tips**: Gunakan gambar berkualitas tinggi dengan resolusi minimal 1920x1080px

### 2. Mengapa Memilih Kami
**Lokasi**: Appearance > Customize > Mengapa Memilih Kami

- Subjudul Section
- Judul Section
- **3 Fitur**, setiap fitur memiliki:
  - Icon (kode Font Awesome, contoh: fa-camera)
  - Judul
  - Deskripsi

**Daftar Icon Font Awesome**:
- `fa-camera` - Kamera
- `fa-award` - Penghargaan
- `fa-heart` - Hati/Cinta
- `fa-star` - Bintang
- `fa-users` - Pengguna/Tim
- `fa-certificate` - Sertifikat
- `fa-thumbs-up` - Jempol
- `fa-shield-alt` - Perisai
- `fa-bolt` - Petir/Cepat
- `fa-gem` - Permata

### 3. Layanan Kami
**Lokasi**: Appearance > Customize > Layanan Kami

- Subjudul Section
- Judul Section
- **3 Layanan**, setiap layanan memiliki:
  - Gambar (Upload gambar 800x600px)
  - Judul
  - Deskripsi

**Tips**: Gunakan gambar landscape dengan rasio 4:3

### 4. Section Blog
**Lokasi**: Appearance > Customize > Section Blog

- Subjudul Section
- Judul Section
- Jumlah Post Ditampilkan (default: 3)

**Catatan**: Post akan otomatis diambil dari blog WordPress Anda

### 5. Testimoni
**Lokasi**: Appearance > Customize > Testimoni

- Subjudul Section
- Judul Section
- **3 Testimoni**, setiap testimoni memiliki:
  - Teks Testimoni
  - Nama Klien
  - Keterangan (contoh: "Prewedding Package - Juli 2023")

### 6. Tombol WhatsApp
**Lokasi**: Appearance > Customize > Tombol WhatsApp

- Nomor WhatsApp (format: 6281234567890, tanpa tanda +)
- Pesan Default WhatsApp
- Toggle Tampilkan/Sembunyikan

## Mengatur Menu

### Primary Menu (Menu Utama)
1. Pergi ke `Appearance > Menus`
2. Buat menu baru atau edit existing menu
3. Pilih "Primary Menu" di bagian "Display location"
4. Tambahkan halaman/link yang diinginkan
5. Klik "Save Menu"

### Footer Menu
1. Ikuti langkah yang sama seperti Primary Menu
2. Pilih "Footer Menu" di bagian "Display location"

## Mengatur Widget Footer

1. Pergi ke `Appearance > Widgets`
2. Anda akan melihat 3 widget area:
   - Footer Widget 1
   - Footer Widget 2
   - Footer Widget 3
3. Drag & drop widget yang diinginkan ke area tersebut

**Widget yang Disarankan**:
- Text Widget (untuk deskripsi perusahaan)
- Navigation Menu (untuk link cepat)
- Custom HTML (untuk informasi kontak)

## Membuat Blog Post

1. Pergi ke `Posts > Add New`
2. Tulis judul dan konten
3. **Set Featured Image** (penting untuk tampilan di homepage):
   - Scroll ke bagian "Featured Image"
   - Klik "Set featured image"
   - Upload gambar 800x600px
4. Publish post

## Membuat Halaman Baru

1. Pergi ke `Pages > Add New`
2. Tulis judul dan konten
3. Publish page

## Tips Optimasi

### Ukuran Gambar yang Disarankan:
- **Hero Carousel**: 1920x1080px (landscape, high quality)
- **Service Images**: 800x600px (landscape)
- **Blog Featured Images**: 800x600px (landscape)
- **Logo**: 200x60px (PNG transparan)

### Optimasi Kecepatan:
1. Compress gambar sebelum upload (gunakan TinyPNG.com)
2. Gunakan format WebP jika memungkinkan
3. Install plugin caching seperti WP Super Cache

### SEO:
1. Install plugin Yoast SEO
2. Isi meta description untuk setiap halaman
3. Gunakan alt text untuk semua gambar

## Upload Logo

1. Pergi ke `Appearance > Customize > Site Identity`
2. Klik "Select Logo"
3. Upload logo Anda (ukuran disarankan: 200x60px)
4. Klik "Publish"

## Troubleshooting

### Gambar Hero Tidak Muncul?
- Pastikan ukuran gambar minimal 1920x1080px
- Upload ulang gambar melalui Customizer
- Clear browser cache

### WhatsApp Button Tidak Bekerja?
- Pastikan nomor dalam format internasional tanpa tanda + (contoh: 6281234567890)
- Cek apakah "Tampilkan Tombol WhatsApp" sudah dicentang

### Menu Tidak Muncul?
- Pastikan menu sudah di-assign ke "Primary Menu" location
- Pastikan menu memiliki minimal 1 item

### Blog Post Tidak Muncul?
- Pastikan post sudah di-publish (bukan draft)
- Cek jumlah post di Customizer > Section Blog
