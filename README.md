# Jaffran Prewed Theme

Tema WordPress elegan dan minimalis untuk fotografer wedding dan prewedding di Bali.

## Fitur Utama

- **Responsive & Mobile-First**: Dibangun dengan Bootstrap 5 untuk tampilan sempurna di semua perangkat
- **Hero Carousel Full Screen**: 3 slide carousel yang dapat dikustomisasi dengan gambar dan teks
- **Sections yang Dapat Dikustomisasi**:
  - Hero Carousel (3 slide)
  - Mengapa Memilih Kami (3 fitur)
  - Layanan Kami (3 layanan)
  - Blog/Tips & Inspirasi
  - Testimoni (3 testimoni)
- **Tombol WhatsApp Floating**: Dapat dikustomisasi dengan nomor dan pesan
- **Desain Elegan & Minimalis**: Fokus pada fotografi dengan layout yang bersih
- **SEO Friendly**: Struktur HTML5 semantic yang baik untuk SEO
- **Optimasi Kecepatan**: Loading cepat dengan optimasi asset

## Instalasi

1. Download tema ini
2. Upload ke WordPress via Appearance > Themes > Add New > Upload Theme
3. Aktifkan tema
4. Pergi ke Appearance > Customize untuk mengatur semua section

## Kustomisasi

Semua section dapat dikustomisasi melalui WordPress Customizer:

### Hero Carousel
- 3 slide dengan gambar, judul, dan subjudul masing-masing
- Teks dan link tombol CTA

### Mengapa Memilih Kami
- 3 fitur dengan icon Font Awesome, judul, dan deskripsi
- Subjudul dan judul section

### Layanan Kami
- 3 layanan dengan gambar, judul, dan deskripsi
- Subjudul dan judul section

### Blog Section
- Jumlah post yang ditampilkan (default: 3)
- Subjudul dan judul section

### Testimoni
- 3 testimoni dengan teks, nama klien, dan keterangan
- Subjudul dan judul section

### WhatsApp Button
- Nomor WhatsApp
- Pesan default
- Toggle show/hide
