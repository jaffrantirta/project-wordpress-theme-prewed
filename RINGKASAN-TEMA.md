## ğŸš€ Langkah Aktivasi Cepat

1. **Aktivasi Tema**:
   - Go to: Appearance > Themes
   - Activate "Jaffran Prewed Theme"

2. **Upload Gambar**:
   - Go to: Appearance > Customize > Hero Carousel
   - Upload 3 gambar (1920x1080px) untuk hero slides
   - Upload gambar untuk services (800x600px)

3. **Atur Konten**:
   - Sesuaikan teks di semua sections
   - Atur icon untuk "Mengapa Memilih Kami"
   - Input testimoni klien

4. **Setup WhatsApp**:
   - Go to: Appearance > Customize > Tombol WhatsApp
   - Masukkan nomor (format: 6281234567890)
   - Atur pesan default

5. **Buat Menu**:
   - Go to: Appearance > Menus
   - Buat menu dan assign ke "Primary Menu"

6. **Publish & Test**!
